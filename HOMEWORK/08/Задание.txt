1) Создать калькулятор(сложение).

На странице два input'a для ввода чисел, один input как кнопка и div. С помощью JS забирать значения из input'ов. Складывать между собой и выводить результат в div.

2) Динамический фон.

Создать массив с цветами (green, blue, orange)
Создать на странице input.
Задача: При вводе числа меняется фон. Число будет подставляться как индекс массива. 
Например в массиве ["green", "blue", "orange"] синий идет под индексом "1".
При вводе числа 1 фон body должен быть синим.

3) Объекты.

Создать объект dog
У объекта будет:
2 свойства(имя собаки(name) и возраст(age)).
2 метода(голос(say) и передвижение(move))

При вызове методов в консоль должно передоваться имя собаки и ее голос. На передвижение имя собаки и текст
dog.say(); // Шарик: Вуф-вуф)
dog.move(); // Собакен Шарик куда-то идет.
